EXD960's project: Kinect to control a robot.

Master thesis project for the robotic's master of University West, Thollättan, Sweden:

Project has been tested on a ABB 1600 1.2 with an IRC5 OS.

Requierement : Kinect v2 up and running.
	       Python v2 installed

The RAPID code need to run before launching the Kinect software.

Only the mimicking mode with the manual tracking input has been tested.



The communication library (abb.py and RAPID code) has been developed by Michael Dawson-Haggerty, his project can be find here : https://github.com/robotics/open_abb.
